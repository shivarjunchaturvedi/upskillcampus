# Banking Information System

## 📌 Project Overview
This project is developed as part of an industrial internship. It is a simple Banking Information System built using Python.

## ⚙️ Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance

## 💻 Technology Used
- Python

## 🚀 How to Run
1. Install Python
2. Run:
   python BankingInformationSystem.py
