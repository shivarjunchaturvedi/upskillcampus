# 🏦 Banking Information System

## 📌 Project Overview
This project is developed as part of an industrial internship under Upskill Campus and UCT. It is a simple Banking Information System built using Python that allows users to perform basic banking operations efficiently.

---

## ⚙️ Features
- ✅ Create Account  
- 💰 Deposit Money  
- 💸 Withdraw Money  
- 📊 Check Balance  

---

## 💻 Technology Used
- 🐍 Python  

---

## 🧠 System Design
The system is designed using basic Python concepts like functions and conditional statements. It works on a console-based interface where users can interact with the system.
